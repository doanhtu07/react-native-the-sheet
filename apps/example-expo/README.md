# example-expo

Example Expo app that demonstrates:

- `react-native-embedded-stack-navigator`

## Running the app

- iOS: `pnpm build:ios`

- Android: `pnpm build:android`
  - Recommend using Java 17
  - If you face network issues on Android, check if you have VPN enabled
    - Try disabling VPN
    - Or wiping the emulator data and rebooting it
