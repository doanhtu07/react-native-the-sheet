## Props

Inherits all props from `Animated.View` except for the following props:

| Prop name  | Type                             | Required | Default     | Description                       |
| ---------- | -------------------------------- | -------- | ----------- | --------------------------------- |
| `onLayout` | `(e: LayoutChangeEvent) => void` | false    | `undefined` | The onLayout callback of the view |
